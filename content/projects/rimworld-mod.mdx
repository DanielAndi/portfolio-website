# Project overview

This mod extends **RimWorld** with custom gameplay elements using the game’s def system, C# assemblies, and **Harmony** for non-destructive patches. Work focuses on balancing new mechanics with vanilla systems and keeping saves compatible across updates.

## Highlights

- **Defs and data-driven content** for items, jobs, and behaviors where possible.
- **Harmony patches** to hook into vanilla logic without forking core code.
- **Testing** in dev mode with logging to catch edge cases early.
