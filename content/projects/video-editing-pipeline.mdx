# Project overview

A **repeatable pipeline** for personal and project video work: standardized ingest, proxies for smooth editing, consistent timeline organization, and export ladders for web and archival masters.

## Highlights

- **Folder templates** and naming so assets stay traceable across revisions.
- **FFmpeg** helpers for transcode, loudness, and batch exports where NLE automation is thin.
- **Preset libraries** for recurring deliverables (e.g. portfolio clips, social cuts).
