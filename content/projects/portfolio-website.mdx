# Project overview

This site is both **the product and the case study**: content-driven project pages, technology filters, skill-based browsing, and a layout tuned for reading and navigation on any screen size.

## Highlights

- **MDX + Contentlayer** for project content with a typed, rebuild-friendly pipeline.
- **Framer Motion** for section and card motion with respect for reduced-motion preferences.
- **Structured frontmatter** (tech stacks, tags, hero galleries) powering cards, filters, and SEO.
